# UPI Transaction Analytics Dashboard | Power BI

## Project Overview

This project is an interactive **Power BI Dashboard** developed to analyze **250K+ UPI transactions** and provide meaningful business insights. The dashboard helps visualize transaction trends, payment behavior, fraud patterns, and customer demographics through interactive reports and KPI monitoring.

---

## Objectives

- Analyze UPI transaction performance.
- Monitor key business KPIs.
- Identify fraud patterns.
- Understand customer demographics.
- Support data-driven decision-making through interactive dashboards.
